
💰 Expense Tracker
🔗 Live Demo: https://yoseph-ad.github.io/Expense-Tracker/
🚀 Overview
The Expense Tracker is a simple and beginner-friendly web application built using HTML, CSS, and JavaScript.
It helps users track their income and expenses, calculate balance automatically, and manage transactions easily.
This project is designed for practicing JavaScript fundamentals and building real-world frontend applications.
✨ Features
💵 Add income and expense transactions
📊 Automatic balance calculation
📈 Separate income and expense tracking
🧾 Transaction history list
❌ Delete any transaction
💾 Data saved using localStorage (persists after refresh)
📱 Simple and responsive UI
🛠️ Built With
HTML5
CSS3
JavaScript (Vanilla JS)
💡 How It Works
Enter a description (e.g. Salary, Food, Transport)
Enter an amount (+ for income, - for expense)
Click Add Transaction
The app updates:
Total balance
Income
Expenses
Transaction list
You can delete any transaction anytime
Data is saved automatically in the browser
